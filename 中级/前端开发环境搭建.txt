1.安装NODEJS
2.验证NODEJS是否安装成功
	npm -v
3.安装angular脚手架
	npm install -g @angular/cli
4.验证angular是否安装成功
	ng -v
5.创建一个前台项目
	新建一个文件夹D:\basic-web
	开始-》运行-》cmd
	d:
	cd basic-web
	ng new basic-web //创建一个angular前台项目

	ng install//安装所需的依赖插件

	启动前台项目
	ng serve --open
